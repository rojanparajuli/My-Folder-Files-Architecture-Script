## Description
A comprehensive script to set up Flutter/Dart projects with Clean Architecture and BLoC pattern.

## Features
- Creates organized project structure
- Sets up Clean Architecture layers
- Configures BLoC state management
- Adds essential dependencies
- Sets up CI/CD workflows
- Configures analysis options
- Creates sample authentication feature

## Usage
```bash
# Create a new Flutter project
flutter create my_project
cd my_project

# Run the setup script
rojanarch

# With options
rojanarch --verbose --ci
rojanarch --help
